# Изображения

Замените плейсхолдеры на реальные фото с сайта ctcs.kz или свои:

- **hero** — главный баннер (склад, контейнеры, логистика)
- **about** — офис, команда или склад
- **services/** — по одному изображению на каждую услугу (аэропорт, фуры, поезд, контейнеры, проект, таможня, склад)
- **contact** — карта или здание офиса

Сейчас используются бесплатные фото с Unsplash для демонстрации. После замены обновите пути в HTML (например с `https://images.unsplash.com/...` на `assets/images/hero.jpg`).
