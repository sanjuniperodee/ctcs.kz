# CTCS Logistics — редизайн сайта

Современная адаптивная версия сайта [ctcs.kz](https://ctcs.kz) с тем же контентом: светлая тема, акцентный цвет, типографика Plus Jakarta Sans, фоновые изображения и карточки с фото.

## Структура

- **Главная** (`index.html`) — hero, блок услуг, преимущества, форма консультации, блок «Получить предложение»
- **О компании** (`about.html`) — описание, миссия, цель, история (таймлайн)
- **Услуги** (`services.html`) — список всех услуг со ссылками на детальные страницы
- **Услуги (детально)** — 7 страниц: авиа, авто, ж/д, мультимодальные, проектная логистика, таможня, обработка и хранение
- **Контакты** (`contact.html`) — офисы (Атырау, Аксай, Уральск, Актау, Алматы), форма консультации

## Как открыть

1. Открыть в браузере файл `index.html` (двойной клик или через меню «Открыть файл»).
2. Либо поднять локальный сервер в корне проекта:
   ```bash
   npx serve .
   # или
   python3 -m http.server 8000
   ```
   и перейти по адресу (например `http://localhost:3000` или `http://localhost:8000`).

## Технологии

- HTML5, семантическая разметка
- CSS: переменные, Grid, Flexbox, медиазапросы, плавные переходы
- Ванильный JavaScript: мобильное меню, обработка формы консультации
- Шрифт: [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) (Google Fonts)

## Адаптивность

- Мобильные: бургер-меню, одноколоночные сетки, увеличенные области нажатия
- Планшеты и десктоп: многоколоночные сетки, фиксированный хедер с размытием

## Изображения на сайте

Сейчас используются **плейсхолдеры с Unsplash** (логистика, офис, карта). Чтобы подставить фото с оригинального ctcs.kz: сохраните нужные изображения в `assets/images/` и замените в HTML ссылки с `https://images.unsplash.com/...` на локальные пути (например `assets/images/hero.jpg`). Подсказки по назначению — в `assets/images/README.md`.
